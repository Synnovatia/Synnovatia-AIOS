# Visual & Brand Style Guide — 2026 Edition

> Governs all Synnovatia visual and brand communications. Source: `Synnovatia_Style_Guide_2026.docx`, imported 2026-07-13. Voice/messaging rules live in `context/brand-voice.md` — this file is the visual counterpart, grounded in the same DiB research.

---

## Why It Changed

The previous palette (Duke Blue #000099 + Kelly Green #339933 + Open Sans) read as a legacy IT-services/nonprofit aesthetic — high-contrast but unsophisticated, undersold the premium advisory product. Client research revealed a gap between how Synnovatia described its value and how clients actually experienced it: authoritative but human, peer-level trust, deep expertise, warmth as a load-bearing differentiator (not incidental).

## Typography

| Role | Typeface | Fallback |
|---|---|---|
| Display / Headlines | **Fraunces** (Google Fonts) — warm, high-character serif | Georgia |
| Body / UI / Subheads | **Barlow** (Google Fonts) — clean humanist sans | Arial (Word docs) |
| Labels / UI Detail | **Barlow Condensed** — uppercase labels, eyebrows, "Messy Middle" label, wordmark lockup | — |

Replaces the earlier Playfair Display + Inter pairing, which read as a default/templated combination rather than a deliberate choice.

**Type scale:**
| Role | Size | Weight | Notes |
|---|---|---|---|
| Display | 40-52px | Regular | Fraunces only, sparingly — hero headlines |
| H1 | 32px | Bold | Fraunces or Barlow — section titles |
| H2 | 24px | Bold | Barlow — subsection headers |
| H3 | 20px | Bold | Barlow, teal color — sub-headers |
| Body | 16-17px | Regular | Barlow, 1.7 line height, generous |
| Small/Caption | 13px | Regular | Barlow — footnotes, tags |
| Label/Eyebrow | 11-13px | Semibold | Barlow Condensed — uppercase, letter-spaced. Use deliberately, eyebrows lose force if overused |
| Pullquote | 18px | Italic | Fraunces italic + gold left-rule — client voice |

**Pull-quote treatment:** Fraunces italic + gold left-rule marks client language as authoritative evidence. Use consistently for testimonials, case-study pull-quotes, and DiB proof points. Example: *"You always leave a session feeling better than when you came in."* — Laura Labovich, The Career Strategy Group

## What Changes, What Stays

**Retire:**
- Kelly Green #339933 and bright blue #0000ff as brand colors
- Pure white #FFFFFF as default page background
- Open Sans as sole typeface
- The Playfair Display + Inter pairing

**Keep & Elevate:**
- Gold #B29200 — now primary accent
- Navy deepened to #0D1F4E
- The Synnovatia wordmark — strong, no change needed
- Fraunces + Barlow + Barlow Condensed type system, anchored to the wordmark
- Arial as the Word-document body fallback
