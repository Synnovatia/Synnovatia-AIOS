# Meeting Summaries

> Fallback/manual archive. Most meetings are now searchable live through the Zoom connector
> (cloud recording + AI summary auto-start as of 2026-07-11) — ask Claude to "find that meeting
> with [name]" and it queries Zoom directly. Use this folder for meetings from *before* that fix,
> or for calls on platforms other than Zoom.

One file per meeting. File format: `YYYY-MM-DD-description.md`

Each file should contain:
- Date, participants, platform
- Key topics discussed
- Decisions made
- Action items (who does what by when)
- Open questions

To add a meeting: paste the summary to Claude and say "save this meeting summary".
